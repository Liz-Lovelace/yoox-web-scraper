Товары распределены по файлам в зависимости от пола и возрастной категории:
  0-24 мес. - baby
  3-8 лет   - child
  9-16 лет  - teen

В каждом файле 100 товаров, для каждого товара есть 4 строки с полями:
  ID товара
  Ссылка на страницу товара
  Ссылка на первую фотографию товара (ботинок сбоку)
  Пустая строка
  
Разрешение фотографий: 1571 x 2000px

На конце ссылок на фотографии есть число "14". 
Если его заменить на число от 13 до 8, то сервер выдаст ту же картинку, 
но с меньшим разрешением, меньше число - меньше разрешение.

Иногда встречаются фото, не совпадающие с форматом "фото правого бока ботинка"
Пример - https://www.yoox.com/images/items/11/11288520nw_14_f.jpg
